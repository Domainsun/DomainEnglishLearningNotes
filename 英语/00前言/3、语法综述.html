              # 五大简单句

              围绕动词

              简单句都是”什么是怎么样“

              ”什么“是主语，人或物

              ”怎么样“ 指 动作，或者发生了什么事

              动作对应动词

              ​

              • 动作可以独立完成（主语 + 不及物动词 = 主谓）
              • 动作有承受者（主语 + 单及物动词 + 宾语= 主谓宾）
              • 动作有两个承受者（主语 + 双及物动词 + 间接宾语 + 直接宾语 = 主谓双宾）
              • 只有一个动作承受者，但句意不完整，需要对这个承受者补充说明（主语 + 单及物动词 + 宾语 + 宾语补语 = 主谓宾补）
              • 非动作（连系动词）连系动词 的作用 就是把 连系动词之后的信息赋值给主语，连接主语和表语。表语也被叫做主语补语，对主语补充说明。（主语 + 连系动词 + 表语 = 主系表）

              构成了五大基本句型

              谓语和谓语动词

              主语后面的成分是谓语。

              谓语动词只是谓语的一部分。

              习惯上把谓语动词称为谓语。

              # 动词的三大本领

              • 表示时间 tense 时
              • 表示状态 aspect 体
              • 表示语气 mood 气

              时间 + 状态 =时态

              时间：现在、过去、将来、过去将来

              状态：一般、完成、进行、完成进行

              4种时间 * 4种状态 = 16种时态

              语气， 不是说话的语气。 是语法中的语气，有

              • 虚拟语气
              • 陈述语气
              • 祈使语气

              需要 助动词 一起协助才能完成，助动词只是帮助谓语动词完成时体气的表达， 没有实际意思。

              协助表示完成时态 ： have

              进行时态 ： be

              有能力 ： can

              有可能：might

              有义务：must

              表示否定 ：do

              被动 ：be

              助动词喜欢伪装， 除了做助动词，还可以做实义动词，有具体的意思表示。

              can ： 易拉罐

              might ： 力量

              do ： 做某事

              have ： 拥有

              be ： 存在

              不要把助动词和它的其他省份混淆！！！

              谓语动词可以可以做谓语动词之外的其他句子成分， 叫做非谓语动词， 只不过要把谓语动词改一下。 改成非谓语动词的形式。

              非谓语动词几乎可以取代所有从句，不过非谓语动词没有动词的时体气三大本领。

              非谓语动词有以下形式：

              • 不定式
              • 现在分词
              • 过去分词
              • 动名词

              现在分词和动名词 和 动词进行时态 长得一样 ， 需要判别是 谓语动词的进行时态 还是 非谓语东的现在分词和动名词

              英语语法的核心就是 动词，能够串起几乎所有的英语语法。

              动词分为

              • 谓语动词

                • 不及物动词
                • 单及物动词
                • 双及物动词
                • 复杂及物动词
                • 连系动词

                决定了对应的五种简单句

                • 主谓
                • 主谓宾
                • 主谓双宾
                • 主谓宾补
                • 主系表

                简单句可以组成

                并列关系的复合句

                从属关系的复杂句

                除了简单句种的5种句子成分，句子成分还有，定语，状语，同位语

                句子成分可以有不同的词性充当

                词性有十大词性

                • 动词
                • 名词
                • 冠词
                • 代词
                • 形容词
                • 数词
                • 副词
                • 介词
                • 感词
                • 连词

                动词除了可以做谓语动词外，可以做其他句子成分，叫做非谓语动词，非谓语动词替代从句，但也就没有了时态和语气的功能。非谓语动词的形式有四种：

                • 不定式
                • 动名词
                • 现在分词
                • 过去分词

              动名次和现在分词 容易谓语动词进行时态混淆。

              英语语法充斥着大量的特例。整体核心简单的总结会牺牲一些细节的准确性。 对整体有认知，而后做积累，改善即可。

              不是学号全套语法才是学会了英语， 掌握核心语法就基本够用了。

              我们不是语言学家， 重要的事用， 不是学和纠结语法细节。
