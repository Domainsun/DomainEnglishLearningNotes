      # Go 环境配置

      # 下载压缩包

      1、 mac 非M1芯片下载 mac 版本的ADM64 tar.gz压缩包，下载地址：https://go.dev/dl/

      2、解压缩到/usr/local：sudo tar -C /usr/local -xzf go1.20.4.darwin-amd64.tar.gz

      3、添加环境配置，编辑~/.bash_profile 文件， 末尾添加以下配置：

      export GOROOT=/usr/local/go
export PATH=$PATH:$GOROOT/bin
export GOPATH=$HOME/go

      4、生效环境配置

      source ~/.bash_profile

      5、开启moudle支持

      go env -w GO111MODULE=on

      6、配置下载依赖镜像 (opens new window)

      go env -w GOPROXY=https://goproxy.cn,direct

      7、查看版本

       go version

      8、查看GOROOT、GOPATH等环境信息

      go env

      9、通过go module模式新建项目

      go mod init hello

      10、编写hello.go文件测试运行

      package main

import "fmt"

func main() {
   fmt.Println("Hello, World!")
}

      通过go run hello.go 运行hello.go 文件
