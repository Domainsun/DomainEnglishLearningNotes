# 谓语

# 预备知识

# 动词的种类

  • 实义动词

    具有明确的动作意义的词，如：“吃”，“打”，“跑”。“喜欢”，“思考”，“认为”类似这样的心里活动的词也是实义动词。举例：eat,hit,run,like,think,consider

    • 短语动词

      实义动词的一种，多个词组成的动词， 一般是实义动词加小品词。短语动词在划分谓语是简单谓语和复合谓语时算一个词。举例：walk down, move down

  • （连系/系)动词

    连接主语和表语，表达主语的状态，构成状态型的复合谓语。举例：be,get,become

  • 助动词

    没有实际意思， 用来帮助动词构成动时态，语气，疑问，否定句。举例：have,do,be,will

  • 情态动词

    用来帮助表达主语的主观态度，看法。举例：can, may,must,shall

# 系表结构和谓语

系表结构是复合谓语的一种， 系动词 + 表语 = 复合谓语。

# be动词

be是系动词的一种，也是最常用的系动词。有8八种变换形式：be,am,is,are,was,were,been,being。

# 什么是谓语

说明主语的动作和状态的成分，表达 主语“怎么样”，主语“是什么”。“怎么样”和“是什么”就是谓语。

# 谓语分类

# 动作型/状态型

“怎么样”类型的就是动作型谓语，“是什么”类型的就是状态型谓语。

  • 动作型
  • 状态型

# 简单谓语/复合谓语

除了按类型划分，还可以按谓语的组成划分。

谓语只有一个部分，就是简单谓语，（实义动词/短语动词） = 简单谓语。

谓语有两个部分，就是复合谓语，(情态动词/助动词 + 动词) / (系动词 + 表语) = 复合谓语

  • 简单谓语
    • 实义动词
    • 短语动词
  • 复合谓语
    • 情态动词 + 动词
    • 助动词 + 动词
    • 系动词 + 表语

# 举例说明

# 动作型谓语

  • He finshed alone in the skiff. 「finish:简单型谓语」
  • I can speak English. 「can speak:复合型谓语」

# 状态型谓语

  • A boy had been with him. 「had been with him : 复合谓语， 系表结构」
  • He was an old man. 「was an old man:复合谓语，系表结构」
